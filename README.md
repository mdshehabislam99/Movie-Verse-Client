# MoviePoka Pro

A comprehensive movie management system where users can browse, manage, and organize their favorite movies with advanced filtering and personal collections.

## Live Demo
[Live Site URL](https://your-domain.netlify.app)

## Key Features
- 🎬 Browse and search through extensive movie database
- ⭐ Create and manage personal movie collections
- 🔍 Advanced filtering by genre, rating, and release year
- 💾 Add custom movies with detailed information
- 🌙 Dark/Light theme toggle
- 📱 Fully responsive design for all devices
- 🔐 Secure authentication with Firebase
- 📝 Real-time CRUD operations
- 🎯 Watchlist functionality for future viewing
- 🎨 Beautiful and intuitive user interface

## Technology Stack
- **Frontend**: React, Tailwind CSS, Firebase Auth
- **Backend**: Node.js, Express, MongoDB
- **Deployment**: Firebase (Client), Vercel (Server)
